# Cybozu2Gcal: Garoon 予定表 → Google カレンダー連携（GAS版）

## 概要

本アプリは、**Garoon（ガルーン）の予定表からエクスポートしたテキスト形式の予定情報を、Google カレンダーに一括登録するためのツール**です。

Google Apps Script（GAS）で実装されており、以下の特長があります。

- ✅ iCal（.ics）ファイルの作成・インポートが不要
- ✅ ブラウザ上のUIだけで完結
- ✅ Google アカウント認証のみで利用可能（APIキー不要）
- ✅ 一方向同期（Garoon → Google カレンダー）
- ✅ 終日予定／時間指定予定の両方に対応

Garoon 側の予定表エクスポート機能と組み合わせることで、  
「エクスポート → 手動変換 → インポート」という手間を大幅に削減できます。

---

## Garoon 側での操作

1. Garoon にログインする
2. **予定表** を表示
3. 対象期間の予定表を **テキストまたはCSV形式でエクスポート**
4. 出力されたテキストをすべてコピー(CSVもテキストとしてコピーする)

※ 本アプリは、以下のような形式のテキストを想定しています。

```
4月2日
10:00-11:00 定例ミーティング
14:00-18:00 作業
```

- 「○月○日」の日付行
- 「開始時刻-終了時刻 件名」の予定行
- 「-----」以降は無視されます

---

## 本アプリの操作方法

### 初期設定（1回のみ）

1. Google Apps Script に本リポジトリのコードをコピー
2. ウェブアプリとしてデプロイ
   - 実行ユーザー：自分
   - アクセス権限：自分のみ
3. 初回アクセス時に Google カレンダーへの認証を許可

---

### 使い方

1. デプロイした Web アプリの URL にアクセス
2. Garoon からコピーした予定表テキストを、テキストエリアに貼り付け
3. **「Google カレンダーに登録」** ボタンをクリック
4. 登録完了メッセージを確認

登録された予定は、Google カレンダー上ですぐに確認できます。

---

## 注意事項

- 同期は **一方向（Garoon → Google カレンダー）** です  
  Google カレンダー側で編集した内容は Garoon には反映されません。
- 同じ内容を何度も実行すると **重複登録** されます  
 （必要に応じて重複防止ロジックを追加してください）
- 年は「実行時の西暦」が使用されます  
 （年跨ぎの予定がある場合は調整が必要です）
